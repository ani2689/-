### 함수

- 선언
    - fun wow(a:Int b:Int) : String
        - wow> 함수의 이름
        - a,b,c>함수의 매개변수(들어올 값)
        - String> 함수의 반환값
        - 함수의 내용은 중괄호({})를 사용하여 서술한다.
            - { return "a = $a, b = $b" }
    - 단일 표현 식 함수
        - 간단한 한 줄 까리 코드를 더욱 보기 쉽게 하자!
        - fun wow(a:Int b:Int) = "a = $a, b = $b"
- 호출
    - wow(5,12)
    - 결과 값 ⇒ a = 5, b = 12
