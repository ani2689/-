### 데이터를 저장하는 방법

- 데이터를 프로그램 내부에 저장하는 방법
    - ex) C언어 내부의 구조체
    - 프로그램이 종료되면 데이터가 삭제된다.
- 파일 시스템을 사용하는 방법
    - ex) C언어의 파일 입출력 사용
    - 프로그램이 종료되어도 데이터가 저장된다.
    - 데이터를 추가, 삭제하기가 쉬워진다.
    - 데이터가 중복될 수 있다.
    - 파일이 프로그램에 종속된다.
- DBMS를 사용하는 방법
    - 종속성과 중복성 문제가 해결된다. 
        - 데이터를 생성하고 싶을 때 DBMS가 데이터를 총합해 생성해준다.
        - User가 SQL 을 사용해 DBMS에 요청한다!
    - 데이터의 일관성이 생겼다.
